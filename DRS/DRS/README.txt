			//File storing
-------------------------------------------------------------------------------------------------------
1. install xampp
2. store the project files inside xampp/htdocs
3. run xampp-control.exe


		//Code running procedure
--------------------------------------------------------------------------------------------------------
1. run xampp-control.exe
2. open command prompt.
3. goto project directory (i.e.  cd xampp/htdocs/DRS)
4.type npm install in command prompt...that will install npm.
5. then type npm update....that will install all the packages to your computer.
6. then type npm start....that will start the npm server.
7. goto your browser and type localhost:8080 in the url to run your project
8. in a new tab type localhost and then click phpmyadmin to go and create your database.


			//Database creation
--------------------------------------------------------------------------------------------------------
1. create a database named "drs"
2. create table name "usres".
